# json-stringify-pretty-compact

forked from [lydell/json-stringify-pretty-compact](https://github.com/lydell/json-stringify-pretty-compact)

`JSON.stringify()` による過度なインデントの挿入を修正し、JSON を "prettify" された文字列として出力する。
```json
{
  "bool": true,
  "short array": [1, 2, 3],
  "long array": [
    {"x": 1, "y": 2},
    {"x": 2, "y": 1},
    {"x": 1, "y": 1},
    {"x": 2, "y": 2}
  ]
}
```
